JUtil by cscott

JUtil is a fully-parameterized (generic) collections library for Java.

JUtil is a fully-parameterized (generic) collections library for Java. It was originally part of the FLEX project. The JUtil library is currently at version 1.4. You can download the latest version, see changes in the latest release, recent changes at ohloh, or browse the API documentation on-line. JUtil is licensed under the GNU GPL.
