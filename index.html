JUtil C. Scott Ananian

A fully-parameterized (generic) collections library for Java.

JUtil was originally part of the FLEX project.

The JUtil library is currently at version 1.4. You can download the latest version, see changes in the latest release, recent changes at ohloh or github, or browse the source code or API documentation. JUtil is licensed under the GNU GPL.

$ git clone git://github.com/cscott/JUtil
